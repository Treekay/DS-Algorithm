# 5. Longest Palindromic Substring  

## Description

Given a string s, find the longest palindromic substring in s. You may assume that the maximum length of s is 

1000.

## **Example1**

```
Input: "babad"
Output: "bab"
Note: "aba" is also a valid answer.
```

## **Rxample2**  

```
Input: "cbbd"
Output: "bb"
```
